HW 8 README

I successfully created the template required for playing a reduced game of scrabble. 
Users can drag and drop tiles onto scrabble board slots and create words to be played. 
As of now, several functions exist within my jquery code that are unfinished. 
This includes the scoring system and the word evaluation functions.
The tiles are functional draggables and the slots/rack all accept tiles into their droppable spaces. 
My css and html are all configured properly and I make use mostly of classes and ids to reference elements.
I struggled to use position tracking effectively, which I believe is the barrier to proper drop registration and error checking. 
Had I more time to improve the positional tracking, i could verify where tiles have been moved to and more readily access their metadata (letter, value, which slot it dropped into)
The word builder is semi functional and can track everything except the REMOVAL of letters from slots. 
This again is something I may have been able to fix with positonal tracking, but was unable to complete in time.
